Blueprint Whiteboard Extractor

Load a .bpex file (extra blueprint data) to extract whiteboard drawings. Each 98,304 bytes = one 384×256 whiteboard. .bp and .bpmeta do not contain the drawing pixels.

Load an image, crop to 384×256 or 256×384. Game uses binary (on/off) only; image is thresholded. Rotate as needed, then replace the selected whiteboard and save.
